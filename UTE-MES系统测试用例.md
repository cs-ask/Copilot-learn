# UTE-MES系统测试用例

## 精磨超段入库检系统测试用例

### 1. 操作工报检

#### 测试场景: 操作工进行入库报检
- **步骤:**  
  1. 操作工登录系统。  
  2. 选择报检模块。  
  3. 输入相关信息（产品编号、检验数量等）。  
  4. 提交报检信息。  
- **预期结果:**  
  1. 系统成功记录报检信息。  
  2. 页面显示报检成功信息。

### 2. 圆度检验

#### 测试场景: 进行圆度检测
- **步骤:**  
  1. 选择圆度检验模块。  
  2. 输入产品信息。  
  3. 选择检测标准。  
  4. 点击开始检测。  
- **预期结果:**  
  1. 系统完成检测，并显示结果。  
  2. 有异常时，系统提示异常信息。

### 3. 光洁度检验

#### 测试场景: 进行光洁度检测
- **步骤:**  
  1. 选择光洁度检验模块。  
  2. 输入产品信息。  
  3. 选择检测方法。  
  4. 点击开始检测。  
- **预期结果:**  
  1. 系统完成检测，并显示结果。  
  2. 有异常时，系统提示异常信息。

### 4. 仓库收洗

#### 测试场景: 仓库物品收洗
- **步骤:**  
  1. 进入仓库收洗模块。  
  2. 输入收洗产品信息。  
  3. 点击收洗按钮。  
- **预期结果:**  
  1. 系统更新库存信息。  
  2. 页面显示收洗成功信息。

### 5. 入库检

#### 测试场景: 进行入库检验
- **步骤:**  
  1. 进入入库检模块。  
  2. 输入入库产品信息。  
  3. 点击开始检验。  
- **预期结果:**  
  1. 系统完成检验，并显示结果。  
  2. 有异常时，系统提示异常信息。
